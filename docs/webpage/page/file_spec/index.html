Specifics about what is inside input/output files.

Files:

  • PW Data File

  • Core Electron Configuration File

  • EXCEED-DM Output File

  • Namelist Input File

  • RHF WF Data File

  • Dielectric Data File

Some of these files will be HDF5 files. The data in these files must be set/read programatically; to handle HDF5 files in python check out, https://docs.h5py.org/en/stable/index.html.
